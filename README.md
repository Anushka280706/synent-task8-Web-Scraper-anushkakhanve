# Web Scraper

A beginner-friendly Python web scraper that extracts structured data from websites using BeautifulSoup and requests. The project scrapes information like titles and prices and stores the extracted data in CSV format.

# Features

* Extract website data
* Scrape titles and prices
* Store data in CSV format
* Uses BeautifulSoup and requests

# Technologies Used

* Python
* BeautifulSoup
* Requests
* CSV

# Run the Project

python web_scraper.py
